# random-joke-generator
Random Joke Generator is a fun web app that fetches a random joke from an API and displays it on the screen. Users can click a button to generate a new joke instantly.
# 😂 Random Joke Generator

A simple web app that displays random jokes using a public API.

## Features
- Generate random jokes
- Simple UI
- Uses JavaScript Fetch API

## Technologies
- HTML
- CSS
- JavaScript

## API Used
https://official-joke-api.appspot.com

## How to Run
1. Clone the repository
2. Open `index.html` in your browser
